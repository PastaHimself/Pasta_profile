About

Clean design. Useful systems. Better game feel.

I build Minecraft Bedrock content with a balance of visual direction and practical gameplay logic, so the result feels sharp without becoming overdesigned.

My work sits between design and implementation. I care about how a project reads visually, how intuitive it feels in play, and how consistent the final experience is from start to finish.

That means thinking about more than just features: model language, texture clarity, interface polish, progression flow, and scripting behavior all need to work together.

Whether I’m building a PvP-focused addon, a server utility system, or a shader pack, the goal stays the same — make it feel clean, modern, and worth coming back to.
